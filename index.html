    Project Ideas

    Nowadays, with the development of technology, people can't live without smart phones any more. Therefore, it is a good chance to develop some new app. App Store is a big online market, there are many different apps can be downloaded. As far as we know, applications were always designed for making people’s life easier. I want to invent a game software for some specific crowd, it can help people decrease their pressure. When they running this game, there will be some bubble swing in the screen, once the user clips the bubble, the music will start to play. People need to set their favourite music in the app.

    I’m a student so I know clearly how much pressure a student will bear in study and it is inevitable. Based on the smartphone has spread to everyone and people will always felt pressure, so I want to develop this game to help them decrease their pressure and relax when they are nervous. As we all know, stress can come from any aspect, it is universal in this society. Therefore, it is necessary to have such a game software in the App Store to help relieve pressure.

    First of all, my project will be simple in visual perception which will be pretty and clean. Because I do not want to users feel more anxious when they start to play this game. The first step to user is set up their personal information, such as gender, ages, position and preferred colour. I will use a monochrome background which is based on the preferred colour that the user has entered in their profile. The reasons why I decided to use monochrome background is because it goes well with bubbles, the stereo sense and shading will work best. Also, user will need to put/download their favourite music in to the library. About this process, I will try to cooperate with some of the popular music software companies, such as QQ music and Spotify, so we can have access to download music from them which is a very important step. I will also put some playlists as advice to people who are not sure what kind of music they want to listen to, it based on their profile like ages and position. After finished setting profile, background and music (it always can be changed in setting), the bubble will start to appear on the screen and the number of bubble are based on the user’s phone type, such as the phone’s screen size, I don’t want to have either too much bobble or not enough. I’m afraid that too many bubbles will cause my project to take up too much CPU space on the user’s phone, which will cause the phone to get stuck and get hot quickly, it is also bad for the phone’s battery. If the bubbles are not enough, even if there is no burden on the phone, the user may just start poking bubbles and end up running out of them quickly. That’s not good for user experience. Once the bubble starts to appear, the game stated. The amount of bubbles is fixed, which means when a bubble is broken, there is a new bubble appear immediately. Therefore, player can prick endless bubbles as they want and their music will start to play since they poked the first bubble. Maybe for the people who just want to try this game, it might make them feel a little boring and stupid. But for the people who are under pressure, is a really good way to relieve their stress, and music can play a good auxiliary effect that make the overall effect of double. The only thing users should pay attention to is not to pop the bubble so hard that the touch of function of the mobile phone screen becomes not sensitive. The project will not assume this responsibility.

    About the tools and technologies, firstly, eclipse is the basic programming application so it has to be required. Also, Android and IOS the two different system in smart phone required different tools and technologies. For Android Eclipse, ADT and SDK is used together for developing. For IOS, XCODE is necessary.

    Doing this project are required rich experience of using smart phone and familiar with everything in the phone. Also, the basic language is necessary in develop a new application, such as C, JAVA and Objective-C, the developer also required familiar with those languages.

    If the project is successful, which means this is good project. It really helped people relieve their stress by their phone. So the aim of this application is achieved. The user information in the database are worked, it suggested the music to the people who have different ages and position. The background correctly run and used the colour which user filled in to the profile. Everything can run in the correct way.
